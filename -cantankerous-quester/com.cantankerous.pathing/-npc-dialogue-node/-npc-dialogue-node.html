NpcDialogueNode

fun NpcDialogueNode(startTile: WorldTile, endTiles: List<Vector3i> = listOf(startTile), doOnce: Boolean, npcName: String, npcAction: String, dialogueSequence: List<Int>, questCompletion: Boolean = false, questAccept: Boolean = false)

Parameters

startTile

The tile near which the NPC is expected to be found.

endTiles

The tiles near which the player is expected to be after interacting with the NPC.

doOnce

True if this action can only be performed once ever by the player.

npcName

The name of the NPC to interact with.

npcAction

The action to apply to the NPC.

dialogueSequence

A list of whole numbers corresponding to the player dialogue options. See the wiki's quest quick guides for examples.

questCompletion

True if a quest completion window will be presented at the end of the dialogue.

questAccept

True if a quest will be offered, and should be accepted, at the end of the dialogue.
