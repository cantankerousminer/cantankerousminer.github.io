class WaypointNode(val endTiles: List<WorldTile>, tolerance: Int = 5) : Node

Walks near the specified tile.

Parameters

endTiles

A list containing the tile to walk near. Only the first tile is used.

tolerance

How close to the tile the player needs to be to consider the tile reached. Values around 4 to 6 often provide for smooth navigation.
