PortalNode

class PortalNode(val startTile: WorldTile, val endTiles: List<Vector3i>, objectName: String, objectAction: String, tolerance: Int = 2, endIsRandom: Boolean = false) : Node

Node allows interactions with options that move the player from startTile to a tile in endTiles such as stairs, ladders, cave entrances, runecrafting guild portal, etc.

Parameters

startTile

The tile near which the portal should be found.

endTiles

A list of tiles near which the player will be located after interacting with the portal.

objectName

The name of the portal object.

objectAction

The action that should be applied to the portal to interact with it.

tolerance

How close to the startTile the portal is expected to be located.

endIsRandom

True of the final destination of the player is not known but expected to be relatively far away from the starting point.
